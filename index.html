Li3 CSS

Lithium library for introducing new CSS features

View the Project on GitHub fahad19/li3_css

  • Download ZIP File
  • Download TAR Ball
  • Fork On GitHub

Brings new features to your CSS files in Lithium framework. Currently works with Library assets only.

Installation

Place li3_css files under app/libraries/li3_css directory, then add this line to app/config/bootstrap/libraries.php

Libraries::add('li3_css');

Examples

Selector Inheritance

.round {
    -moz-border-radius: 5px;
    -webkit-border-radius: 5px;
    border-radius: 5px;
}

#test-round {
    extend: .round;
}

becomes

.round {
    -moz-border-radius: 5px;
    -webkit-border-radius: 5px;
    border-radius: 5px;
}

#test-round {
    -moz-border-radius: 5px;
    -webkit-border-radius: 5px;
    border-radius: 5px;
}

The extend property may also contain multiple selectors separated by a comma.
